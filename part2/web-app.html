Чем отличаются Web server и Application server?

Основное различие между веб-сервером и сервером приложений заключается в том, что веб-сервер предназначен для обслуживания статических страниц, например HTML и CSS, в то время как сервер приложений отвечает за генерацию динамического контента путем выполнения кода на стороне сервера, например JSP, сервлета или EJB.
